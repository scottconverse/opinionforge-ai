OpinionForge

An editorial craft engine for opinion writing with precise rhetorical control

What It Is

OpinionForge is an AI-powered editorial craft engine for generating publication-ready opinion pieces — op-eds, columns, and long-form opinion content. It is built around 12 distinct rhetorical modes that define argumentative approach, prose style, and rhetorical devices.

OpinionForge gives you three independent controls over how your piece is written:

  • Mode — the rhetorical approach (e.g., polemical, analytical, satirical)
  • Stance — argumentative emphasis direction from -100 (equity-focused) to +100 (liberty-focused)
  • Intensity — rhetorical heat from 0.0 (measured) to 1.0 (maximum conviction)

This is a tool for improving your own rhetoric — not for imitating anyone else's. Choose a mode based on what the argument demands, not who you want to sound like.

Installation

pip install opinionforge

Requires Python 3.11+. Configure your API key before using:

export ANTHROPIC_API_KEY=your-key-here
# or for OpenAI:
export OPENAI_API_KEY=your-key-here
export OPINIONFORGE_LLM_PROVIDER=openai

Quick Start

# Generate with a specific rhetorical mode
opinionforge write "The rise of algorithmic governance" \
  --mode polemical \
  --no-preview

# Choose stance and intensity
opinionforge write "Climate policy tradeoffs" \
  --mode analytical \
  --stance -20 \
  --intensity 0.4

# Blend two modes
opinionforge write "Technology regulation" \
  --mode satirical:60,forensic:40 \
  --no-preview

# Export to Substack format
opinionforge write "The attention economy" \
  --mode analytical \
  --export substack \
  --no-preview

The 12 Rhetorical Modes

Each mode is a self-contained rhetorical system with its own prose patterns, argument structure, and signature devices. Use opinionforge modes --detail <mode-id> for full details on any mode.

Confrontational

--mode polemical

Polemical

confrontational

Combative moral urgency. Names targets, indicts positions as contemptible, demands the reader choose sides.

--mode populist

Populist

confrontational

Plain-spoken, us-versus-them. Amplifies ordinary voices against remote, impersonal power.

--mode provocative

Provocative

confrontational

Deliberately unsettling. States unfashionable truths and forces readers to face uncomfortable implications.

Investigative

--mode forensic

Forensic

investigative

Evidence-first. Assembles documentary proof, follows the money, builds a prosecutorial case from primary sources.

--mode data_driven

Data-Driven

investigative

Quantitative. Leads with numbers, visualizes trends, anchors every claim in empirical precision.

Deliberative

--mode analytical

Analytical

deliberative

Rigorous, evidence-first. Breaks complex issues into parts, weighs explanations, arrives at measured conclusions.

--mode dialectical

Dialectical

deliberative

Thesis-antithesis-synthesis. Takes opposing arguments seriously and arrives at positions through genuine engagement.

--mode measured

Measured

deliberative

Calm, earned authority. Conveys confidence through restraint and precision. Claims only what the evidence warrants.

Literary

--mode satirical

Satirical

literary

Irony, exaggeration, comic incongruity. The argument is made by letting the target's logic destroy itself.

--mode oratorical

Oratorical

literary

Grand, elevated. Deploys the full arsenal of classical rhetoric — anaphora, periodic sentences, tricolon.

--mode narrative

Narrative

literary

Story-first. Uses scene, character, and testimony to make abstract arguments vivid and emotionally resonant.

--mode aphoristic

Aphoristic

literary

Compressed, epigrammatic. Delivers argument through maxims and paradoxes. Every word carries maximum weight.

Controls: --mode, --stance, --intensity

OpinionForge separates what rhetorical approach to use from how to position the argument and how forcefully to argue it:

Flag Range Default Description
--mode mode ID or blend analytical Rhetorical approach. Single mode or colon-separated blend (e.g., polemical:60,analytical:40)
--stance -100 to +100 0 Argumentative emphasis: -100 (equity-focused) to +100 (liberty-focused). 0 = balanced
--intensity 0.0 to 1.0 0.5 Rhetorical heat: 0.0 = measured and deliberative, 1.0 = maximum conviction 
opinionforge write "Immigration and labor markets" \
  --mode forensic \
  --stance -40 \
  --intensity 0.7 \
  --no-preview

Export Formats

Use the --export flag to produce platform-optimized output:

  • --export substack — Clean ATX markdown, no HTML tags
  • --export medium — Medium-style markdown with DROP CAP and story formatting
  • --export wordpress — Gutenberg block markup for direct paste
  • --export twitter — Numbered tweet thread with 280-character limits
opinionforge write "The state of journalism" \
  --mode oratorical \
  --export substack \
  --no-preview

Disclaimer Policy

Every piece generated by OpinionForge includes a mandatory disclaimer that cannot be suppressed:

This piece was generated with AI-assisted rhetorical controls. It is original content and is not written by, endorsed by, or affiliated with any real person.

This policy exists because OpinionForge generates original AI content with a defined rhetorical character. The disclaimer ensures readers understand what they are reading. It is not negotiable.
